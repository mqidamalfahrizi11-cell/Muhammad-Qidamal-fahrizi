
# Contoh dan Fungsi setiap tag
tag <html> berfungsi sebagai tag pembuka dalam membuat dokumen HTML.
tag <head> memuat informasi meta tentang dokumen, seperti judul.
tag <title> berfungsi untuk menetapkan judul dari halaman web.
tag <body> mendefinisikan body atau isi dari halaman web dan berisikan seluruh konten yang ditampilkan.
tag  sampai  merupakan jenis tag heading yang berfungsi untuk membuat heading pada halaman.
tag <hr> berfungsi untuk membuat garis horizontal pada halaman, biasanya untuk memisahkan konten.
tag <p> berfungsi untuk mendefinisikan paragraf.
tag <b> berfungsi untuk membuat teks bold (tebal).
tag <i> fungsinya untuk membuat teks italic (miring).
tag <u> berfungsi untuk menggaris bawahi teks.
