# Muhammad-Qidamal-fahrizi

1. <h1> sampai <h6>
Fungsi: Membuat judul atau heading. <h1> adalah yang terbesar dan utama, sedangkan <h6> adalah yang terkecil.
Contoh: <h1>Selamat Datang di Website Saya</h1>

2. <p>
Fungsi: Digunakan untuk membungkus teks dalam bentuk paragraf.
Contoh: <p>Ini adalah teks paragraf pertama saya.</p>

3. <a> (Anchor)
Fungsi: Membuat hyperlink untuk menghubungkan satu halaman ke halaman lain atau URL eksternal.
Contoh: <a href="https://google.com">Klik di sini untuk ke Google</a>

4. <img>
Fungsi: Menampilkan gambar di halaman web. Tag ini tidak memiliki tag penutup (self-closing).
Contoh: <img src="gambar.jpg" alt="Deskripsi Gambar">
5. <ul> dan <li>
Fungsi: <ul> untuk membuat daftar list poin (bullet), dan <li> adalah item di dalamnya.
